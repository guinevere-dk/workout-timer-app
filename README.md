# Workout Timer App

조용하게, 짧게, 시간을 기반으로 운동 루틴을 재생해주는 타이머 앱입니다.

## Features (v1.0)

- 샘플 루틴 3개 제공 (첫날 5분, 전신 10분, 걷기+달리기 12분)
- 커스텀 루틴 생성 / 편집 / 삭제
- 운동 기록 저장 및 “운동 기록” 화면에서 전체 내역 확인
- 오늘 운동 시간, 최근 루틴, 캐릭터 메시지 표시

## Tech Stack

- React Native
- TypeScript
- AsyncStorage (로컬 데이터 저장)
